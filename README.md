Featured Project: Global Superstore Dashboard

**Overview:**  
Analyzed sales, profit, discounts, and regional trends using the Global Superstore dataset.  
Data was cleaned and prepared in Python, then visualized in Looker Studio for interactive exploration.

Tools Used:
- Python (Pandas, NumPy)
- Looker Studio (Google Data Studio)
- Excel (for data validation)
  
Key Insights:
- Technology and Office Supplies are the top-performing categories.  
- Discount rates negatively affect profit margins.  
- Regional analysis shows potential in emerging markets.
